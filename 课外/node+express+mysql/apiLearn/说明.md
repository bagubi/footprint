### Express API 学习指南目录
1. 创建 Express 项目
2. 基础项目结构
3. 创建简单的服务器文件 (server.js)
4. 配置 package.json 脚本
5. 运行项目
# 创建项目目录
mkdir express-api-learn
cd express-api-learn

# 初始化 npm 项目
npm init -y

# 安装 Express
npm install express

# 安装开发依赖（可选）
npm install --save-dev nodemon
### 基础项目结构
apiLearn/
├── app.js                           # 主应用入口文件，配置Express应用和中间件
├── package.json                     # 项目配置文件，包含依赖、脚本和元数据
├── 说明.md                          # 项目说明文档，包含学习指南和结构说明
├── util/                            ########## 工具类目录，存放通用工具函数
│   ├── dbconfig.js                  # 数据库连接池配置，管理MySQL连接
│   └── responseHandler.js           # 统一响应处理工具，标准化API返回格式
├── models/                          ########## 数据模型层目录，处理数据访问逻辑
│   └── youyuModel.js                # youyu表数据模型，封装数据库操作方法
├── routes/                          ########## 路由层目录，处理URL路径映射
│   ├── index.js                     # 主路由文件，处理根路径和通用路由
│   ├── users.js                     # 用户相关路由，处理用户功能接口
│   └── youyu.js                     # youyu相关路由，处理youyu表API接口
├── controllers/                     ########## 控制器层目录，处理业务逻辑
│   └── youyuController.js           # youyu控制器，协调路由和模型之间的交互
├── views/                           ########## 视图层目录，存放模板文（Jade模板）
│   ├── index.jade                   # 首页模板
│   ├── error.jade                   # 错误页面模板
│   └── layout.jade                  # 页面布局模板
└── public/                          ########## 静态资源目录，存放前端静态件
    ├── stylesheets/                 # CSS样式文件
    │   └── style.css                # 默认样式文件
    ├── javascripts/                 # JavaScript文件
    └── images/                      # 图片资源


# 进入项目目录
cd express-api-learn

# 启动项目
npm start
