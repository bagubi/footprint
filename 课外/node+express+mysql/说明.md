### Express API 学习指南目录
1. 创建 Express 项目
2. 基础项目结构
3. 创建简单的服务器文件 (server.js)
4. 配置 package.json 脚本
5. 运行项目
# 创建项目目录
mkdir express-api-learn
cd express-api-learn

# 初始化 npm 项目
npm init -y

# 安装 Express
npm install express

# 安装开发依赖（可选）
npm install --save-dev nodemon
### 基础项目结构
express-api-learn/
├── package.json
├── server.js
└── routes/
    └── api.js

# 进入项目目录
cd express-api-learn

# 启动项目
npm start
