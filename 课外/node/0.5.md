# URL 中的端口号

## 什么是端口号？
端口号是标记服务器上运行的特定服务或应用程序的**数字标识符**。  
它允许服务器区分不同的服务，并将请求路由到正确的应用程序。

## 端口号范围
| 范围 | 类型 | 说明 |
|------|------|------|
| `0 - 1023` | 知名端口（Well-known Ports） | 预留给系统级服务（如 HTTP、FTP、SSH） |
| `1024 - 49151` | 注册端口（Registered Ports） | 可被用户或应用程序注册使用 |
| `49152 - 65535` | 动态/私有端口（Dynamic/Private Ports） | 通常用于临时连接 |

## URL 结构示例

| 组成部分 | 说明 | 示例 |
|--------|------|------|
| 🔑 协议（钥匙） | 数据传输规则 | `http://` |
| 🏠 域名（地址） | 服务器位置 | `localhost` |
| 🚪 端口号（房间号） | 服务入口 | `3000` |
| 📁 资源路径（物品位置） | 具体文件或接口 | `/index.html` |

## 默认端口
- **HTTP** 默认使用端口：`80`  
  → `http://example.com` 等价于 `http://example.com:80`
- **HTTPS** 默认使用端口：`443`  
  → `https://example.com` 等价于 `https://example.com:443`

> 💡 当使用默认端口时，浏览器会自动省略端口号。
